⚡ Mapa de Componentes
Pilar 1: Percepción 360° (33%)
∿

Calidad de Señal (Osciloscopio)

Puntaje por competencia, brechas entre perspectivas, consistencia entre evaluadores. Onda limpia = buena percepción; ruido = puntos ciegos.

Pilar 2: Disciplina Gerencial (33%)
⚛

Integridad del Circuito (Trazas)

Evaluaciones a tiempo, cumplimiento 1-a-1, planes de desarrollo, horas de capacitación. Trazas sólidas = disciplina; corrosión = procesos descuidados.

Pilar 3: Impacto en el Equipo (34%)
☀

Rendimiento de Carga (LED)

Rotación voluntaria, engagement/eNPS, promociones internas, reconocimientos. LED brillante = alto impacto; LED tenue = bajo impacto.

Métricas Derivadas del Circuito
V

Voltaje = Pilar 1 + Pilar 2

Percepción + Disciplina impulsan el sistema. Fuerza motriz del circuito.

I

Corriente = Pilar 3 (Impacto)

Flujo de energía que el equipo genera. Impacto en personas = corriente activa.

R

Resistencia = Inverso de Pilar 2

Baja disciplina = alta resistencia = pérdida de energía. R = V / I

C

Capacitor = Pilar 2 + Pilar 3

Capacidad de desarrollo de talento derivada de disciplina e impacto combinados.

T

Transformador = Score Global

Transforma el input en output. Eficiencia refleja el Score de Liderazgo combinado.

♨

Calor = Burnout (P = I² × R)

Resistencia alta + corriente forzada = energía desperdiciada, desgaste organizacional.

⚠

Fusible = Cultura Organizacional

Protege el sistema. Se quema si calor > umbral y disciplina < 30.
